# FrontEnd Parking

## Como iniciar o front end:

### Passo a passo:

#### Passo 1:

Clone o repositório:

```
git clone https://github.com/renanrider/desafio-entrevista-nodejs.git
```

#### Passo 2:

Acesse o diretório:

```
cd .\desafio-entrevista-nodejs.git\
```

#### Passo 3:

Instale as dependências:

```
npm install
```

#### Passo 4:

Instale as dependências:

```
ng serve
```

E navegue até:

```
http://localhost:4200/
```
